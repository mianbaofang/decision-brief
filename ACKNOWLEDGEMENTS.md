# Acknowledgements / 致谢

## Open-source dependencies

- FastAPI and Starlette: backend web framework.
- Uvicorn: ASGI server.
- HTTPX: HTTP client and testing support.
- Pydantic: data validation.
- SQLite: local-first decision archive storage.
- pytest and pytest-asyncio: test framework.
- Playwright: browser UI testing.
- Edge TTS / `edge-tts`: text-to-speech support.

## References and inspiration

- [jinchenma94/bazi-skill](https://github.com/jinchenma94/bazi-skill): BaZi-related inspiration.
- [op7418/humanizer-zh](https://github.com/op7418/humanizer-zh): Chinese anti-AI-writing style reference.
- Browser Web Speech API: speech input support.
- Amap Open Platform: weather context service used by the Nature lens.
- OpenAI-compatible LLM ecosystem: optional user-configured decision-brief generation.
